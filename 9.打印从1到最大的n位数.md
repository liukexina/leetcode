# 打印从1到最大的n位数

## 题目
输入数字 n，按顺序打印出从 1 到最大的 n 位十进制数。比如输入 3，则打印出 1、2、3 一直到最大的 3 位数 999。

示例：
```
输入: n = 1
输出: [1,2,3,4,5,6,7,8,9]
```

## 解答
```js
var printNumbers = function(n) {
let str = '';
    const arr = [];
    for(let i = 1 ; i <= Math.pow(10, n) - 1 ; i++) {
      arr.push(i)
    }
    return arr
};
```

## 思路
例如最大的 11 位数是 99 ，最大的 22 位数是 9999 ，最大的 33 位数是 999999 。

则可推出公式：end = 10^n - 1
